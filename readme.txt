== Description ==

Плагин создания каталога товаров для магазина зарегистрированного в сервисе АвтоОфис: http://autoweboffice.com/

== Installation ==

= Устанвока =

1. Загрузите плагин "AutoWebOffice Catalog  Of Goods" (АвтоОфис - Каталог товаров) в ваш блог
2. Активируйте загруженный плагин
3. В правом вертикальном меню выберите пукт "Каталог товаров"
4. Из выпадающего списка меню плгина выберите раздел "Настройки"
5. Укажите настройки подключения к вашему магазину в сервисе "АвтоОфис": http://autoweboffice.com/
6. Сохраните введенные данные
7. После ввода данных для подключения к магазину, в разделе "Настройки" нажмите на кнопку "Обновить товары"
8. Дождитесь завершения обновления
9. Для размезения каталога товаров у вас на сайте на нужной вам странице вставьте тег [show_autoweboffice_catalog_of_goods]

Если появятся вопросы или предложения по улучшению плагина, отправляйте их на почту zakaz@autoweboffice.com

= Рекомендации =

* PHP 5.1 или выше
* WordPress 3.5.1 или выше
